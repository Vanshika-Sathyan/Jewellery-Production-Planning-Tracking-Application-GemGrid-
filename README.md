GemGrid, the Jewellery Production Planning System is a web-based system designed to assist in managing and tracking production orders for different stages of production. It is a system that can assist production managers in tracking production order progress. The system provides a facility for tracking production order stages in real-time and can calculate stage capacity. The system can assist in production planning for a jewellery production environment.

Technology Stack ->
Frontend:
-	React.js
-	Material UI
-	Axios

Backend:
-	Node.js
-	Express.js

Database:
-	MongoDB Atlas

Authentication:
-	JWT (JSON Web Token)
-	bcrypt for password hashing

Version Control:
-	Git and GitHub
